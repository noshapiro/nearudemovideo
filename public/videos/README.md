# Video assets for HR Onboarding comparison

Place the following files in this directory:

- **hr-standard.mp4** — Agent A (Without NEARU) demo
- **hr-nearu.mp4** — Agent B (With NEARU) demo

These are shown in the video comparison section when the "HR Onboarding" scenario is selected in Quick Tools.

If you already have them in `public/video/`, you can copy or move them here so the paths `/videos/hr-standard.mp4` and `/videos/hr-nearu.mp4` resolve correctly.
